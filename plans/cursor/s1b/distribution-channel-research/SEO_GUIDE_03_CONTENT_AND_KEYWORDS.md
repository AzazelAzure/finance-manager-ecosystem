# SEO Guide 03 — Content Strategy & Keywords for a PH PFM

**Purpose:** Keyword research, content planning, and ongoing optimization strategy tailored to our niche: a Personal Finance Manager (PFM) targeting Filipino households. This is the guide you revisit each sprint to keep SEO momentum.

**Sprint link:** Reference this when planning blog content, landing page copy, or feature naming. Every new public-facing page should target at least one keyword cluster from §2.

---

## 1. Our SEO Positioning

### 1.1 The Niche Advantage

We are *not* competing with Mint, YNAB, or generic global budgeting apps. We are competing for:
- **Filipino-specific** search queries
- **Mobile wallet-native** (GCash/Maya) feature queries
- **Tagalog/Taglish** search terms that global competitors completely ignore

This is our moat. No global PFM app optimizes for "paano mag-budget gamit GCash" or "expense tracker para sa Pinoy." The search volume is lower, but we can own the entire results page.

### 1.2 Competitor Landscape (PH PFM SEO)

| Competitor | SEO Strength | Gap We Can Exploit |
| --- | --- | --- |
| **MoneyMax.ph** | Strong content, high domain authority | Aggregator, not a tool — they compare, we *do* |
| **GCash/Maya blogs** | Brand authority | They don't teach budgeting — they promote features |
| **Bluecoins / Money Manager** | App store SEO only | Zero web content strategy — no blog, no guides |
| **r/phinvest** | Community SEO (Reddit) | User-generated, not product — we can create authoritative guides |

## 3. Content Calendar Template

SEO content must be **consistent**. One well-researched article per month is better than five rushed ones.

### Recommended Cadence (Aligned to Dev Timeline)

| Phase | Content Type | Frequency | Focus |
| --- | --- | --- | --- |
| **T-12 to T-6 Mo** (Alpha) | Educational blog posts | 2/month | Clusters 3, 5 (build authority, no product push) |
| **T-6 to T-3 Mo** (Beta) | Feature-focused content | 2/month | Clusters 1, 2 (show the product solving real problems) |
| **T-3 to T-0 Mo** (Launch) | Comparison / review content | 1-2/month | Cluster 1 + competitor comparisons |
| **Post-Launch** | User stories + advanced guides | 1-2/month | All clusters + new long-tail discoveries |

### Blog Post Template Structure

Every blog post should follow this SEO-optimized structure:

```
1. Title (contains primary keyword, < 60 characters)
2. Meta description (contains keyword, compelling CTA, < 155 characters)
3. Introduction (hook + keyword in first 100 words)
4. H2 sections (each targets a related keyword)
5. Practical examples with PH-specific context
6. Internal link to the app / another blog post
7. CTA (soft — "Try Finance Manager free" or similar)
```

### Example Blog Post Ideas (Ready to Write)

1. **"5 Budget Apps for Filipinos in 2026 — Which One Actually Works?"**
   - Cluster 1 + comparison intent
   - Compare ourselves honestly against Bluecoins, Money Manager, Fortune City
   - Include GCash/Maya integration as our differentiator

2. **"How to Track Your GCash Spending (Without Going Crazy)"**

## 4. On-Page Optimization Checklist

Run this checklist for **every public page** before it goes live:

- [ ] **Title tag** contains primary keyword and is < 60 characters
- [ ] **Meta description** is < 155 characters, contains keyword, has a CTA
- [ ] **H1** is unique, contains the primary keyword, and there is exactly ONE per page
- [ ] **H2/H3** headings use related keywords naturally
- [ ] **URL slug** is short, descriptive, and contains the keyword (e.g., `/blog/gcash-expense-tracker`)
- [ ] **Images** have descriptive `alt` text (not "image1.png")
- [ ] **Internal links** connect to at least 2-3 other pages on the site
- [ ] **External links** reference 1-2 authoritative sources (e.g., BSP, DTI)
- [ ] **Canonical URL** is set via `<link rel="canonical">`
- [ ] **Mobile rendering** looks correct (test on actual phone)
- [ ] **Page load time** is < 3 seconds on 3G (test via PageSpeed Insights)

## 5. Link Building Strategy (Off-Page SEO)

Backlinks are hard to get organically, especially for a new brand. Here's a realistic strategy:

### 5.1 Organic / Free Methods
- **Reddit** — Genuinely helpful answers on r/phinvest with a subtle product mention
- **Facebook Groups** — Share educational content (not product links) that drives traffic to blog posts
- **Guest posting** — Offer to write for Filipino personal finance blogs (they need content)
- **HARO / journalist requests** — Answer journalist queries about PH fintech

### 5.2 Earned Methods
- **Press releases** — When launching publicly, submit to PH tech news (e.g., When In Manila, Rappler Tech)
- **Product Hunt** — Launch on PH-specific collection
- **App review sites** — Submit to MoneyMax.ph, Philstar lifestyle, etc.

### 5.3 What NOT to Do
- ❌ Buy backlinks (Google penalizes this)
- ❌ Spam comments on blogs with your link
- ❌ Create fake review sites linking back to you
- ❌ Use link exchange schemes

## 6. Ongoing SEO Maintenance (Per Sprint)

Add this to every feature sprint that touches public-facing pages:

### Sprint SEO Checklist
- [ ] Check Google Search Console for new crawl errors
- [ ] Review search queries — are we ranking for anything unexpected?
- [ ] Update `sitemap.xml` if new public routes were added
- [ ] Run PageSpeed Insights on any modified public pages
- [ ] Check for broken internal links
- [ ] Review and refresh meta descriptions if CTR is low (< 2%)

### Quarterly SEO Review
- [ ] Re-research keyword clusters — any new long-tails emerging?
- [ ] Audit top 5 blog posts — can they be updated with fresher data?
- [ ] Check competitor movement — is anyone new ranking for our keywords?
- [ ] Review backlink profile — any toxic links to disavow?
