# T03.SL3: Dashboard HelpModeWrapper A11y

## Scope
- **Surface**: Web `HelpModeWrapper`
- **Focus**: Accessibility and z-index overlap issues.

## Execution Checklist
- [ ] Ensure that `HelpModeWrapper` clickable regions are accessible via keyboard navigation (focus).
- [ ] Verify that `z-index` contexts behave correctly (e.g., the Joyride overlay should sit above the modal layer, which sits above the page layer).

## Verification
- Tabbing through the Dashboard with Help Mode enabled should highlight the wrappers.
- The Joyride tooltip must render visibly on top of other dashboard elements without getting clipped by `overflow: hidden` containers.
